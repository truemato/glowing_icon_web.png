# glowing_icon_web.png

URL:https://truemato.github.io/glowing_icon_web.png/

HDRを画像に埋め込んで光らせる\

PNGに Rec2020/PQ (HDR) ICCプロファイルを埋め込みつつ、背景色に近いピクセルだけを倍率で調整して
「ブラウザで見ると発光して見える」画像を作ります。\
X とか Instagram で周りより光って見える画像を見たことがありませんか? あれです。

# Use

PNGファイルを選択するか、ドラッグ&ドロップします

# Option

背景色をスポイトして background R,G,B に指定してください。\
tolerance が小さいほど背景だけに影響します。\
strength でその明るさを調整します。 

弱めたい: strength 0.85～0.95

強めたい: strength 1.05～1.15（上げすぎると白飛びしやすい）
